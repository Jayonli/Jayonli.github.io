picgo

发表于2020-10-09|更新于2020-10-29|Tools
|阅读量:

一、了解Picgo

一个用于快速上传图片并获取图片 URL 链接的工具。

PicGo 本体支持如下图床：
七牛图床 v1.0
腾讯云 COS v4\v5 版本 v1.1 & v1.5.0
又拍云 v1.2.0
GitHub v1.5.0
SM.MS V2 v2.3.0-beta.0
阿里云 OSS v1.6.0
Imgur v1.6.0

本体不再增加默认的图床支持。你可以自行开发第三方图床插件。详见 PicGo-Core官方指南

二、配置到vscode

1.我使用的图床：SMms,到这注册，获得 Secret Token
2.Vs code搜索安装Picgo插件
3.将Smms获得的Token填入 picgo>Pic Bed>Smms：Token 下的框中即可。

三、快捷使用

剪贴板：ctrl+alt +u
文件夹:ctrl +alt +e
指定路径:ctrl +alt +o

**注意:**若是无反应，查看是否与vscode其他快捷键有冲突，或者与本地其他软件的全局快捷键有从图。
