Valine 底部出现Code 403
发表于2020-10-24|更新于2020-10-29
issues|valine•403
项目场景 部署到阿里的新域名后，在Valine评论框后出现Code 403: Access denied by API domain white list就出现了。 原因分析之前用的是GitHub的域名[用户名.github.io],后面更换了阿里的新域名，没有在leancloud的valine应用中重新添加新的域名。 解决方案 登入:[leancloud的国际版(]https://console.leancloud.app/#/apps) 进入当初设置的valine应用，点击侧边栏设置，选择【安全中心】，在【Web 安全域名】中，换行添加新的域名即可！ 然后就可以正常显示了，nice…..
GitHub出现Commit failed
发表于2020-10-23|更新于2020-10-30
issues|github•tools
问题描述：windows中，使用GitHub Desktop对github项目进行push。出现如下提示： 1Commit failed - exit code 1 received 原因分析：因为这个项目是在github上clone下来修改的，现在提交到我的仓库，但项目还包含着之前别人的github仓库信息，信息在.git文件夹。 解决方案：把隐藏文件显示出来，然后删除隐藏文件夹: .git 。 win10操作:查看–>选项–>查看–>勾选“显示隐藏的文件…” 删除隐藏的 .git 文件夹就行！
